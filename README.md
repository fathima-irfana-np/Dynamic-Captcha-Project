This project introduces a Dynamic Cognitive CAPTCHA system that leverages AI-driven logic, story-based animations, and visual reasoning to provide human-only verification challenges. It offers a secure, interactive, and intelligent alternative to traditional CAPTCHA methods.

## Features

- Animated scene-based challenges
- Context-aware questions linked to each story
- AI-powered (planned) dynamic story and question generation
- Detection of bot-like behavior through interaction patterns
- Adaptive difficulty based on failed attempts
- Accessibility-conscious design

## File Structure

├── backend/
│ ├── app.py # Flask/Django backend logic
│ ├── logic.py # CAPTCHA logic and answer validation
│ └── data/ # Story templates and questions
├── assets/ # Visual or animation assets

└── README.md # Project documentation



## Tech Stack

- Frontend: HTML5, CSS3, JavaScript (Canvas API or Lottie)
- Backend: Python with Django or Flask
- Planned AI: OpenAI GPT models, HuggingFace Transformers
- Data: JSON-based story templates and logs
